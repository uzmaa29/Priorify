# Priorify
A simple task prioritization app built from a real personal problem.

### ✦ About Prorify

Priorify is a beginner-friendly task prioritization web application designed to help users organize tasks based on priority, urgency, and completion status.

I built this to solve a personal problem, I often felt confused about which task to do first when everything felt important. Priorify helps by clearly showing tasks and their priorities in one place and organises it with clarity.

### ✦ Features

Add tasks with: </br>
Title </br>
Description </br>
Priority (Low / Medium / High) </br>
Due date </br>
Mark tasks as completed </br>

Dashboard showing: </br>
Total tasks </br>
Active tasks </br>
Completed tasks </br>
Urgent tasks </br>

Clean and simple user interface </br>
